Repository of MIW1 phylogenetic  tree in angiosperms

Phylogenetic Analysis with IQ-TREE: TPR Protein Families This repository contains the input data and output results of a Maximum Likelihood phylogenetic inference performed using the IQ-TREE software. The analysis was based on a multiple sequence alignment of proteins containing tetratricopeptide repeats (TPRs).

Analysis Details Program: IQ-TREE version 3.1.2

Command executed: iqtree -s mafft_TPRs_v3.1.fas -m MFP -bb 1000 -nt AUTO

Input data: Protein alignment in FASTA format.

Alignment dimensions: 74 sequences and 1397 columns (1223 distinct patterns, 815 parsimony-informative sites).

Model selection: ModelFinder Plus (-m MFP). It evaluated multiple models to find the best evolutionary fit (e.g., Q.PLANT+I+G4).

Support Test: Ultrafast Bootstrap with 1000 replications (-bb 1000).

File Index: The purpose of each file generated and stored in this repository is described below:

Input Data: mafft_TPRs_v3.1.fas: Original multiple alignment file in FASTA format generated by MAFFT.

Main Results (Trees): mafft_TPRs_v3.1.fas.treefile: Final Maximum Likelihood phylogenetic tree in Newick format. This is the main file you should use to visualize the topological tree with the original branch lengths.

mafft_TPRs_v3.1.fas.contree: Consensus tree built from the bootstrap replications. It displays the support values ​​at the nodes.

Reports and Information from the Run: mafft_TPRs_v3.1.fas.iqtree: Main report, user-friendly and easy to read. It contains a summary of the analysis, the selected evolutionary model, a plain text representation of the tree, and the substitution rate parameters.

mafft_TPRs_v3.1.fas.log: Complete log of the entire IQ-TREE process on the terminal. Useful for debugging or reviewing computation times and search convergence.

Secondary Files / Intermediate Data: mafft_TPRs_v3.1.fas.bionj: Initial tree constructed using the Neighbor-Joining (BIONJ) method, which served as the starting point for Maximum Likelihood optimization.

mafft_TPRs_v3.1.fas.mldist: Matrix containing the genetic distances of pairs estimated by Maximum Likelihood.

mafft_TPRs_v3.1.fas.splits.nex: NEXUS format file containing the information on the splits of the bootstrap trees.

mafft_TPRs_v3.1.fas.model.gz: Compressed file with a detailed log of the likelihoods of all evolutionary models tested during the ModelFinder phase.

mafft_TPRs_v3.1.fas.ckp.gz: Checkpoint file. Allows resuming the analysis from where it left off if it was interrupted.

How to view the results To view the tree files (.treefile or .contree), you can download the files and open them with any standard phylogenetic tree viewer, such as:

FigTree (Desktop)

iTOL - Interactive Tree of Life (Web)

Dendroscope (Desktop)
