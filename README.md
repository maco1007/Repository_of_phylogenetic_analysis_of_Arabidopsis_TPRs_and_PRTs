Phylogenetic Analysis of At5g17270 Homologs in Angiosperms

This repository contains the input data and output results of a Maximum Likelihood phylogenetic inference performed using the IQ-TREE software.

The analysis focused specifically on the identification and evolutionary reconstruction of homologs of the protein encoded by the At5g17270 locus (a protein containing TPR repeats) across 59 angiosperm taxa.

Analysis Details
Software: IQ-TREE version 3.1.2

Command executed: iqtree -s mafft_TPRs_v3.1.fas -m MFP -bb 1000 -nt AUTO

Input data: Multiple sequence alignment in FASTA format.

Alignment dimensions: 74 sequences (representing homologs identified in the 59 angiosperm taxa) and 1397 columns (1223 distinct patterns, 815 parsimony information sites).

Model Selection: ModelFinder Plus (-m MFP). Evaluated multiple models to find the best evolutionary fit for this family (e.g., Q.PLANT+I+G4).

Support Test: Ultrafast Bootstrap with 1000 replicates (-bb 1000).

File Index
The purpose of each file stored in this repository is described below:

Input Data
mafft_TPRs_v3.1.fas: Original multiple alignment file in FASTA format generated by MAFFT containing the 74 homologous sequences of At5g17270.

Main Results (Trees)
mafft_TPRs_v3.1.fas.treefile: Final Maximum Likelihood phylogenetic tree in Newick format. This is the main file for visualizing the topology and estimated branch lengths for homologues.

mafft_TPRs_v3.1.fas.contree: Consensus tree constructed from bootstrap replications. It displays the support values ​​(UFBoot) at the nodes to assess clade confidence.

Run Reports and Information
mafft_TPRs_v3.1.fas.iqtree: Detailed main report. It contains the analysis summary, the optimal evolutionary model selected for the set of homologues, a plain text representation of the tree, and the substitution rate parameters.

mafft_TPRs_v3.1.fas.log: Complete log of the entire IQ-TREE process performed by the terminal. Useful for verifying computation times and search convergence.

Secondary Files / Intermediate Data
mafft_TPRs_v3.1.fas.bionj: Initial tree constructed using the Neighbor-Joining (BIONJ) method, used as a starting point for optimization.

mafft_TPRs_v3.1.fas.mldist: Genetic distance matrix calculated by Maximum Likelihood between the sequenced homologs.

mafft_TPRs_v3.1.fas.splits.nex: NEXUS format file containing information on the splits derived from the bootstrap.

mafft_TPRs_v3.1.fas.model.gz: Compressed file containing the log-likelihood values ​​for all models tested by ModelFinder.

mafft_TPRs_v3.1.fas.checkpoint.gz (or .ckp.gz): Checkpoint file that allows you to resume the analysis if it is interrupted.

How to view the results
To explore and edit the phylogenetic trees (.treefile or .contree), it is recommended to download the files and use standard visualization platforms such as:

iTOL - Interactive Tree of Life (Web-based, highly recommended for large trees)

FigTree (Desktop software)

Dendroscope (Desktop software)
